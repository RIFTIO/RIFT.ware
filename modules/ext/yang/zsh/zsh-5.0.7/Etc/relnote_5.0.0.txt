Version 5.0.0 of zsh is the new stable release.  There are many changes
since the last stable release series, 4.2, of which the most important
is handling of multibyte characters.  The file NEWS in the distribution
gives more information.  The file README details some incompatibilities
with previous versions, mostly minor.

Users of the development release series will not see major changes since
the previous release, 4.3.17.
